\section{Additional Analysis: Synchronization Dynamics}
\label{sec:additional}

We analyze \emph{synchronization dynamics}: how the model identifies the active component and when this precedes belief refinement.

\paragraph{Motivation.} The model faces two tasks with different computational requirements: meta-synchronization (comparing accumulated statistics via likelihood ratios in log-probability space) and within-component belief tracking (constrained Bayesian updates~\cite{piotrowski2025constrained}). We probe each independently.
