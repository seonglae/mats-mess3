\section{Additional Analysis: Synchronization Dynamics}
\label{sec:additional}

We chose synchronization dynamics because it directly probes the hierarchical structure of in-context learning: does the model first identify the source, then refine beliefs?

\paragraph{Layer-wise probing.} Figure~\ref{fig:layer_probe} shows held-out $R^2$ for meta-belief and within-component belief probes at each layer. Within-component belief is already $R^2 = 0.86$ at embedding and reaches $0.99$ at L0. Meta-belief (component identity) starts near zero and builds progressively: $0.00 \to 0.48 \to 0.86 \to 0.93 \to 0.94$ across embed$\to$L0$\to$L1$\to$L2$\to$L3.

\begin{figure}[h]
\centering
\includegraphics[width=0.48\textwidth]{p2_layer_probing.png}
\caption{Within-component belief (red) is nearly perfect from L0; meta-belief (black) builds progressively across layers.}
\label{fig:layer_probe}
\end{figure}

This is surprising: we predicted component identification before belief refinement (Prediction 3), but the model maintains parallel belief tracks for all components, with the correct track becoming dominant as meta-synchronization sharpens. This is consistent with the constrained belief update: attention corrections are valid regardless of which component is active.

\paragraph{What we would do with more time.} (1) Causal interventions: perturb the component-identity direction and verify factored independence. (2) Scaling with $K$: test whether dimensionality follows $K{+}1$. (3) GHMM extension: add Bloch Walk components via belief-state-based generation. (4) Training dynamics: track when separated vs.\ factored geometries emerge, testing whether the FWH attractor applies to block-diagonal structure.
