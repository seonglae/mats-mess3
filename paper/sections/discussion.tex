\section{Discussion}

\paragraph{Non-ergodicity forces hierarchical inference.} The model cannot achieve low loss without first identifying the source. This mirrors how LLMs must identify the active regime before predicting well within it.

\paragraph{Block-diagonal structure.} The joint operator is block-diagonal, not tensor-product, placing our setup outside the standard FWH setting. Whether the factoring inductive bias extends to this case is an open question that our experiments address.

\paragraph{Residual stream as belief space.} The residual stream has no privileged basis; geometric arrangement encodes prediction similarity. If the model discovers that source identity and within-source state are independent dimensions of prediction, it should allocate orthogonal subspaces accordingly.

\paragraph{Limitations.} Small model (3--4 layers); static components (no drift or switching); same token alphabet across components. Natural extensions include slowly drifting parameters, GHMM components such as Bloch Walk (which would require belief-state-based generation rather than hidden state simulation), and connections to induction heads~\cite{olsson2022context} as a mechanism for source discrimination.
