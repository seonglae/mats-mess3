\section{Discussion}

\paragraph{Non-ergodicity forces hierarchical inference.} The model cannot achieve low loss without first identifying the source. This mirrors how LLMs must identify the regime (topic, style, task) before predicting well.

\paragraph{Block-diagonal vs tensor-product.} The joint operator is block-diagonal, not tensor-product, so standard FWH does not directly apply. Yet we observe partial factoring, suggesting the factoring inductive bias extends beyond conditionally-independent processes and acts as a ``representational attractor.''

\paragraph{Residual stream as belief space.} The residual stream has no privileged basis; geometric arrangement encodes prediction similarity. Factored structure means the model discovered that source identity and within-source state are independent dimensions of prediction, allocating orthogonal subspaces.

\paragraph{Limitations.} Small model (3--4 layers); static components (no drift/switching); same token alphabet across components. Extensions: slowly drifting parameters, shared/component-specific tokens, connections to induction heads~\cite{olsson2022context}.
