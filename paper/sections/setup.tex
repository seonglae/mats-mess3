\section{Non-Ergodic Mess3 Mixture}

\subsection{Construction}

We construct $K$ Mess3 processes with distinct parameters $\{(\alpha_k, x_k)\}_{k=1}^K$. Each training sequence is generated entirely by one component chosen uniformly; the model observes only tokens, not the component identity.

We choose components with well-separated entropy rates $H_k$ and subdominant eigenvalues $\zeta_k = 1-3x_k$:

\subsection{Relevance to Language Models}
\label{sec:relevance}

This structure mirrors natural language in three ways. \textbf{ICL as source discrimination}: the model must use early tokens to identify the generating process, analogous to topic/style adaptation. \textbf{Covariate shift}: different components have different token distributions, requiring in-context adaptation. \textbf{Hierarchical latent structure}: component identity is a slow variable (constant per sequence), hidden state is fast (evolves per token), mirroring the multi-scale structure of language.

\subsection{Experimental Setup}

\paragraph{Model.} GPT-2 style decoder-only transformer via TransformerLens~\cite{nanda2022transformerlens}. Phase 1 (single Mess3): 3 layers, $d_{\text{model}}=64$, 4 heads. Phase 2 (mixture): 4 layers, $d_{\text{model}}=128$, 4 heads. Pre-norm, learned positional embeddings, vocabulary $\{$BOS, 0, 1, 2$\}$.

\paragraph{Training.} Next-token CE loss, Adam ($\text{lr}=5\times10^{-4}$, no weight decay). Fresh batches of 8192 sequences generated on-the-fly. Context: 16 tokens + BOS. Phase 1: 10K steps, Phase 2: 5K steps.

\paragraph{Baselines.} Bayes-optimal CE at each position via Monte Carlo: both \emph{oracle} (knows component) and \emph{Bayesian} (infers component from observations).

\paragraph{Analysis.} (1) CEV via PCA, (2) linear regression to belief states ($R^2$), (3) meta-belief regression, (4) per-position/layer probing for synchronization dynamics, (5) attention patterns.
