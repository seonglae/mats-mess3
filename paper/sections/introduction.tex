\section{Introduction}

Language models face non-ergodic data: different documents constitute distinct statistical regimes. In-context learning requires identifying the active regime before predicting optimally. We study this in a controlled setting using Mess3 processes \cite{marzen2017nearly, shai2024transformers}, 3-state nonunifilar HMMs whose belief geometry forms a Sierpinski gasket.

By constructing a non-ergodic training set where each sequence comes from one of $K$ Mess3 parameterizations, we create a minimal in-context learning problem: the transformer must determine which process generated the sequence and track its hidden state. This connects to the Factored World Hypothesis (FWH) \cite{shai2026factored}: if the model treats component identity and within-component belief as separate factors, it should represent them in orthogonal subspaces with dimensionality $(K{-}1) + 2 = K{+}1$, rather than the joint $3K{-}1$. Crucially, our non-ergodic mixture has \emph{block-diagonal} transition operators (not tensor-product), so the standard FWH does not directly apply. This creates an interesting intermediate case: does the factoring inductive bias extend beyond conditionally-independent processes? The FWH predicts that transformers first learn factored representations, expanding dimensionality only under sustained gradient pressure \cite{shai2026factored}. Our setup tests whether this pattern holds for block-diagonal structure.

\paragraph{Contributions.} (1) We formalize the non-ergodic Mess3 mixture as hierarchical belief inference and derive geometric predictions, identifying three possible geometries (\S\ref{sec:predictions}). (2) We verify single-process gasket recovery ($R^2 = 0.989$) with loss matching the Bayes-optimal predictor, then analyze the mixture (\S\ref{sec:results}). (3) We analyze synchronization dynamics: how quickly the model identifies the active component across context positions and layers (\S\ref{sec:additional}).
